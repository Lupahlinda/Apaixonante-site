# 🏇🏿Apaixonante Site🐴 - 🐴 Projeto de Cavalos Lendários 🏇🏿

Um projeto pessoal desenvolvido para exibir informações sobre cavalos lendários das corridas japonesas, além de um perfil pessoal interativo.

## 📋 Descrição do Projeto

Este projeto é uma coleção de páginas HTML que demonstram:
- Perfil pessoal com links e navegação
- Galeria de cavalos lendários com informações detalhadas
- Uso de tags HTML semânticas e formatação
- CSS minimalista e responsivo

## 🚀 Como Visualizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Lupahlinda/Apaixonante-site.git
   ```

2. Navegue até a pasta:
   ```bash
   cd Apaixonante-site/atv02
   ```

3. Abra qualquer arquivo HTML no navegador:
   - `perfil.html` - Página principal
   - `cavalos.html` - Galeria de cavalos
   - `pagina.html` - Página simplificada

## 🔧 Personalização

### Para adicionar seu próprio perfil:
1. Edite `perfil.html`
2. Substitua `me.jpg` por sua foto
3. Atualize links de redes sociais
4. Modifique informações pessoais

### Para adicionar mais cavalos:
1. Adicione novas imagens na pasta
2. Edite `cavalos.html`
3. Siga o padrão existente:
   ```html
   <div class="cavalo">
       <img src="novo_cavalo.jpg">
       <p>Nome do Cavalo (ANO)</p>
   </div>
   ```
