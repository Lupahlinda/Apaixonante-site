# 🏇🏿Apaixonante Site🐴 - 🐴 Projeto de Cavalos Lendários 🏇🏿

Projeto pessoal desenvolvido para exibir informações sobre cavalos lendários das corridas japonesas, além de um perfil pessoal interativo.

## 📋 Descrição do Projeto

Este projeto é uma coleção de páginas HTML. Ele demonstra o uso de tags HTML semânticas e formatação, além de apresentar uma galeria de cavalos lendários e um perfil pessoal.

## 🌐 Páginas do Site

1. [perfil.html](perfil.html) - Página principal com perfil pessoal, links e navegação.
2. [cavalos.html](cavalos.html) - Galeria completa dos cavalos lendários com informações detalhadas.
3. [pagina.html](pagina.html) - Página simplificada com duas fotos.

## 🎨 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessibilidade.
- **CSS3**: Estilização minimalista e responsiva.
- **Git**: Controle de versão.
- **GitHub**: Hospedagem do projeto.

## 🚀 Como Visualizar

1. Clone o repositório: `git clone https://github.com/Lupahlinda/Apaixonante-site.git`.
2. Abra qualquer arquivo HTML no navegador.
